## Arbeit ohne Callback

```js
let a = [1, 2, 3];
let result = [];

for (i = 0; i < a.length; i++) {
  result.push(a[i] + 10);
}

result; // => [11, 12, 13]

```

---
## Arbeit mit Callback

```js
let a = [1, 2, 3];
let result = [];

a.forEach(function(item){
  result.push(item + 10)
})

result; // => [11, 12, 13]

```
